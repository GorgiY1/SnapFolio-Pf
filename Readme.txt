Modern Landing Page Template — HTML/SCSS/JS
What's inside:
index.html

css/style.css

js/scripts.js

php/mail.php

How to use:
Open index.html in your browser

To edit texts, open the file in a code editor (e.g. VS Code)

To change the logo — replace the logo.png file in the /img folder

For the contact form to work — open php/mail.php and:

enter your email address, or

set up SMTP

Hosting:
You can upload the site to any hosting provider, or use Netlify / GitHub Pages (note: PHP won't work on GitHub Pages)
